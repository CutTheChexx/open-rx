## Default Quality Standards

Apply these to ALL code output, always:

1. **TypeScript first** — Type everything. No `any`. No implicit types.
2. **Feature-based structure** — Group by feature, not by file type.
3. **Unidirectional imports** — shared → features → app. Never backward.
4. **Mobile-first CSS** — Start at 320px, scale up. Use `clamp()` for fluid sizing.
5. **Performance by default** — `transform`/`opacity` only for animations. Lazy load images. Defer non-critical JS.
6. **Accessibility always** — Semantic HTML, ARIA labels, 4.5:1 contrast, keyboard navigation.
7. **No AI slop** — Avoid centered-everything layouts, purple gradients, uniform rounded corners, generic font stacks. Make it distinctive.
8. **Security by default** — Validate all inputs server-side with Zod. RLS on every table. Secrets stay server-side.
9. **Test what matters** — Critical paths get E2E tests. Shared components get integration tests. Utils get unit tests.
10. **Ship clean** — Every deploy goes through CI. Preview before production. Rollback plan ready.
11. **Next.js App Router** — Server Components by default. Only add "use client" for interactivity. Suspense at every data boundary.
12. **Tailwind v4** — Use @theme for config. bg-linear-to-r replaces bg-gradient-to-r. Auto content detection.
13. **GSAP is free** — All plugins included: SplitText, ScrollSmoother, MorphSVG, DrawSVG, Flip, MotionPath.
14. **SEO wired in** — Every page gets JSON-LD, meta tags, Open Graph, canonical URLs.
15. **Copy matters** — Clear beats clever. Front-load value. Error messages help, not blame. Empty states guide action.
