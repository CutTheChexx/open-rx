# Open RX (OTC) — Free Tier

**The Prescription.** 7 skills. 4 reference files. Zero fluff.

Open RX is a Cowork / Claude Code plugin that makes Claude actually good at building production web apps. No more generic boilerplate. No more AI slop. Every skill is packed with real patterns from real codebases.

```
/plugin install open-rx
```

---

## What's Included (Free)

| Skill | What It Does |
|-------|-------------|
| **rx-writer** | The consultation system. Describe what you're building, get a full technical prescription — stack, schema, file structure, implementation order. Start here. |
| **react-architecture** | Feature-based project structure, compound components, custom hooks, error boundaries. From bulletproof-react patterns. |
| **nextjs-patterns** | App Router done right. Server Components, Suspense boundaries, Server Actions, caching, streaming. |
| **performance-ops** | Core Web Vitals optimization, bundle splitting, lazy loading, LCP/INP/CLS patterns. |
| **prompt-engineering** | System prompt architecture, XML patterns, chain of thought, structured output. Claude 4.6 optimized. |
| **copy-craft** | UX writing, headlines, CTAs, error messages, empty states, email copy. |
| **local-seo** | JSON-LD structured data, service area pages, city page architecture, meta tags. |
